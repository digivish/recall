# Skill: Nestor System Auditor
## Objective
Ensure 100% alignment between the Postgres Schema, the API routes, and the UI.

## Verification Rules
1. **Data Integrity:** Do the form fields in `onboarding` and `journal` match the Prisma columns exactly?
2. **Design Integrity:** Are we using Tailwind variables for our grayscale palette, or are there 'rogue' hex codes?
3. **Route Integrity:** Does every button in the `/design` folder have a corresponding valid route in `/src/app`?
